# Front-End Opposites

A front-end visual experiment built as part of a personal portfolio, focused on contrast, opposites, and expressive UI composition in a modern browser-based format.

## Live Demo

https://laurandreea10.github.io/Front-End-Opposites/

## Features

* Contrast-driven UI concept
* Visual exploration of opposites
* Interactive front-end presentation
* Modern layout and styling
* Responsive design
* GitHub Pages deployment

## Technologies

* HTML
* CSS
* JavaScript
* GitHub Pages

## Purpose

This project was created as a personal front-end experiment exploring visual contrast, interface composition, and creative browser-based design.
